**Praxion 0.1 – USB Malware Scanner & Cleaner** is a powerful, lightweight, and cross-platform Python-based tool that automatically scans every USB drive you plug in, detecting and removing malware in real time. It uses multi-layer security techniques like YARA rule matching, heuristic scanning, behavior analysis, and PE file inspection to identify both known and unknown threats. Suspicious files are safely quarantined, detailed forensic logs are maintained, and user-friendly reports explain detections clearly. Praxion runs silently in the background, provides clear notifications, and supports optional integrations like VirusTotal and ssdeep hashing. With one-command setup and zero configuration, it’s easy to use while offering deep educational insights into malware detection. Perfect for students, IT admins, and cybersecurity researchers, Praxion provides enterprise-grade protection and peace of mind against infected USB drives.
